# Local Services Booking Platform 🚀

A full-stack MERN application that connects customers with local service providers such as electricians, plumbers, painters, and cleaners.

## Features

### Customer
- Register and login
- Browse services
- Search service providers
- Book services
- View bookings
- Add reviews
- Favorite providers

### Provider
- Create and manage services
- Update and delete services
- Manage availability
- Accept or update bookings
- Manage profile

## Tech Stack

Frontend:
- React.js
- Vite
- React Router
- Axios
- Context API

Backend:
- Node.js
- Express.js

Database:
- MongoDB
- Mongoose

Authentication:
- JWT
- Refresh Token
- bcryptjs

## Security
- Role-based authorization
